`Irasutoya` is ruby library and CLI tool for [いらすとや](https://www.irasutoya.com)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'irasutoya'
```
## Usage

```ruby
Irasutoya.random #=> Returns a random image from irasutoya
```
